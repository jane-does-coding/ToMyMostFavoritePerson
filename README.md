## Setup

### 1. Clone the repo

```bash
git clone https://github.com/jane-does-coding/ToMyFavoritePerson
cd to-my-favorite-person
```

### 2. Install the dependencies

```bash
npm i
```

### 3. Run the project

```bash
npm run dev
```
